# Directory Admin Guide

Accessible from the main admin dashboard by clicking into any directory.

## Directory Dashboard

From `/directories`, click any directory to enter its dashboard. Manage everything specific to that directory:

### Listings
- View, add, edit, delete business listings
- Set business type (local, supplier, distributor, farm, etc.)
- Manage claim status (claimed, unclaimed, pending)
- Upload business images

### SEO Articles
- Generate AI-optimized Answer-First articles
- Track impressions, clicks, keyword rankings
- Manage article sponsorships (business subscribers)

### Blog
- Write and publish blog posts with templates
- Set post type (blog, community, article)
- Community posts appear on the public community page

### Categories
- Organize listings with hierarchical categories
- Businesses can be tagged with multiple categories

### Monetization
- Create plan tiers (Listed, Featured, Premium)
- Set monthly/yearly pricing and grandfathered rates
- Create sponsored article packages and ad zones

### Deals
- Create deals with templates (Percentage Off, Fixed Price, BOGO, Free Gift)
- Set redemption limits, start/end dates
- Track redemptions by code

### Call Tracking
- Assign trackable phone numbers to listings
- View call logs and duration data

### Analytics
- Track page views, listing clicks, deal claims, engagement trends

### Integrations
- Core Swift tenant setup (one-click)
- External CRM connections (HubSpot, Salesforce, Pipedrive, Zoho)
- Webhook configuration

### Branding
- Custom colors, logos, favicon, custom domain mapping

### Trap Doors
- Build hyper-niche landing pages for long-tail SEO
- Programmatic page generation from templates

### Settings
- Directory name, slug, city metadata, SMTP, domain settings

### Workflows
- Browse and install automation workflow templates
