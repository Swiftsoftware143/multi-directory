📂 Multi-Directory Platform

User Guide v2.0

1. Introduction

The Multi-Directory Platform is a full-stack white-label directory SAAS. Create and manage multiple city-specific or industry-specific directories from a single dashboard. Each directory has its own public URL, theme, SEO settings, blog, newsletter, deals, CRM, call tracking, monetization, and more.

🏭 Multi-Industry Templates

10 pre-built templates: local business, farm, restaurant, real estate, medical, service, education, automotive, fitness, hospitality.

🎨 Customizable Themes

Brand colors, fonts, logos, layout per directory. Each directory looks unique.

📱 City-Based URLs

Each directory gets its own public URL at /{slug}. Auto geo-redirect for visitors.

🧮 AI Content Engine

Blog generator, business articles, programmatic pages, content repurposing.

📊 Analytics & CRM

Track views, clicks, calls, visitors. Manage contacts, pipelines, and deals.

💰 Monetization

Tiers, subscriptions, ad zones, sponsored listings, article sponsorships.

43. Admin Panel vs Directory Dashboard

This platform has two layers of navigation. Think of it like a mall:

🏢 Global Admin (The Mall)

  • Dashboard — cross-directory stats (foot traffic across stores)
  • Directories — create, manage, clone, sell directories
  • API Keys — Google, OpenAI, Stripe (fuels all directories)
  • Settings — tenant name, password

This is where you walk between directories and manage the building.

🏪 Directory Dashboard (Each Store)

  • Dashboard — directory stats (listings, articles, posts)
  • Listings + Categories — manage businesses
  • Blog + SEO Articles (AI Writer) — content marketing
  • SEO + Trap Doors + Branding — rankings & appearance
  • Monetize — tiers, subscriptions, ad zones, sponsored articles
  • Analytics — per-directory views, clicks, events
  • Integrations — Core Swift, CRM, webhooks, SMTP
  • Settings + Legal + Domains — directory config

Everything the directory owner needs in one place.

How to Enter a Directory Dashboard

  1. Go to Directories in the global admin
  2. Click the 📂 Dashboard button on any directory card
  3. The sidebar swaps to that directory's controls
  4. Click the back arrow to return to global admin

What Happens When You Sell a Directory

The buyer logs into their own dashboard and sees only their directory — with full access to listings, SEO Articles, monetization, analytics, branding, integrations, and SMTP. They connect their own Core Swift tenant, their own CRM, their own sending email. Everything is independent.

4. Directories & Cities

Each directory represents a city or industry. Directories use a city-based URL structure. The root domain is a homepage city grid; each directory lives at /{slug}.

🏠 Root Homepage

The root domain shows a city grid. Visitors browse available cities.

🏡 Per-City Pages

/{city-slug} (e.g., /palm-bay). Displays businesses, deals, blog posts, categories.

🌍 Geo-Redirect

IP-based detection offers to redirect visitors to their nearest city directory. Non-blocking, dismissible banner.

🔍 Per-City SEO

Unique meta title, description, OG tags, JSON-LD schema, canonical URL, sitemap entry per city.

49. FAQ

How do I create a new directory?

From the Dashboard, click New Directory. Select a template, enter your city/industry name, and choose a URL slug. The directory will be created with default settings and categories matching the template.

Can I change a directory's template after creation?

Yes. Go to the directory settings page and select a new template. Note that custom field values from the old template that don't exist in the new one will be preserved in the database but hidden from the public view.

How do I add businesses in bulk?

Use the CSV Import feature under the Import/Export section. Download the sample CSV template, fill in your business data, and upload. The system validates and imports with a detailed log.

What AI content features are available?

The platform offers AI blog generation, business article generation, programmatic page generation, newsletter content generation, and content repurposing. Each feature has directory-level configuration for tone, length, and topic focus.

How do I set up call tracking?

Navigate to Call Tracking in the admin panel. Provision a local phone number from available inventory, assign it to a business, and configure forwarding. Calls are automatically logged and recorded.

Can I use my own domain?

Yes. Register your domain under Domains in the admin panel. Follow the verification instructions (DNS TXT record or file upload). Once verified, assign the domain to any directory. SSL is automatic.

How does monetization work?

Define subscription tiers with different feature sets and pricing. Businesses can subscribe to premium tiers for enhanced listings, ad zones, and sponsored placements. Invoices are generated automatically and payouts are tracked per directory.

How do I integrate with n8n?

Create an API key under Settings > API Keys. Use the webhook system to send events to n8n endpoints. The platform exposes all CRUD operations via REST API for n8n HTTP Request nodes to consume.

Is there a mobile app?

The platform is fully responsive and works on mobile browsers. There is no native mobile app, but the admin dashboard and public directory pages are optimized for mobile viewing.

How do I get support?

Contact your platform admin for technical issues, account questions, or feature requests. Include your account email and a description of the issue.
